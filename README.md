# PXG Rainbow Orb Helper

Projeto criado com React para ajudar no acompanhamento e copia das coordenadas dos Rainbow Orbs.

As informacoes das localizacoes dos Rainbow Orbs foram pegas do Discord oficial do jogo.

O progresso fica salvo localmente para permitir continuar depois de onde parou.

Esse projeto foi 100% vibe coded e esta aberto a PRs de contribuicao para melhorias, correcoes e novas ideias.
